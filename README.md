## NETSTAT
NETSTAT (Network Statistics) is a command-line tool used to display information about the network connections, routing tables, interface statistics, and open ports of a computer. It helps users and network administrators monitor and troubleshoot network-related problems.

## ipconfig
IPCONFIG (Internet Protocol Configuration) is a command-line tool used in Windows to display and manage the IP address and network configuration of a computer. It helps users view and troubleshoot network settings.

## ping
PING (Packet Internet Groper) is a command-line network utility used to test the connectivity between two devices on a network. It checks whether a host (computer/server) is reachable and measures the time taken for data packets to travel from the source to the destination and back.

## tracet
TRACERT (Trace Route) is a command-line network diagnostic tool used to find the path that data packets take from a source computer to a destination computer on a network. It also shows the time taken at each step (hop) along the route.

NSLOOKUP (Name Server Lookup) is a command-line network tool used to find the IP address of a domain name or the domain name of an IP address using the DNS (Domain Name System).

GETMAC is a command-line tool in Windows used to display the MAC (Media Access Control) address of network adapters on a computer. The MAC address is a unique hardware identifier assigned to each network device.
